# Prevención de Riesgos Laborales (PRL)

## Obligación legal
Si tienes **1 o más empleados** → obligatorio tener plan de prevención.
Si estás sola como autónoma → muy recomendable (proteges tu salud y evitas multas).

## Riesgos principales en taller
| Riesgo | Causa | Prevención |
|--------|-------|------------|
| Golpes/atrapamientos | Elevador, vehículo mal sujeto | Revisión elevador, calzos, protocolo |
| Cortes | Herramientas manuales | EPIs (guantes anticorte), orden |
| Quemaduras | Motor caliente, soldadura | Guantes térmicos, señalización |
| Intoxicación | Gases (CO, humos) | Ventilación forzada, extractor |
| Contacto eléctrico | Baterías, cortocircuitos | EPIs dieléctricos, formación |
| Caídas | Suelo resbaladizo | Limpieza, calzado antideslizante |
| Sobreesfuerzos | Levantar piezas pesadas | Carro elevador, técnicas posturales |
| Exposición química | Aceites, disolventes, líquidos | Guantes nitrilo, mascarilla, ventilación |

## EPIs obligatorios
- [ ] Calzado de seguridad (puntera metálica)
- [ ] Guantes de trabajo (mecánico)
- [ ] Guantes de nitrilo (químicos)
- [ ] Gafas de seguridad
- [ ] Ropa de trabajo adecuada (no ropa suelta)
- [ ] Protectores auditivos (si hay mucho ruido)

## Equipos de protección colectiva
- [ ] Extintor (revisado, en sitio visible y accesible)
- [ ] Mantas ignífugas
- [ ] Señalización de emergencia
- [ ] Botiquín completo
- [ ] Lavaojos de emergencia
- [ ] Ficha de datos de seguridad de cada producto químico

## Documentación obligatoria (si tienes empleados)
- [ ] Plan de prevención escrito
- [ ] Evaluación de riesgos (firmada por técnico PRL)
- [ ] Registro de formación a trabajadores
- [ ] Reconocimientos médicos anuales
- [ ] Investigación de accidentes (si ocurren)

## Dónde contratar servicio PRL
- **Mutua colaboradora con la SS** (mutua de accidentes: FREMAP, IBERMUTUA, etc.)
- **Servicio de prevención ajeno** (SPA): Asepeyo, Quirón Prevención, etc.
- Coste: ≈200-400 €/año para autónoma sin empleados
